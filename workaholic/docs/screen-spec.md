## 全域共用元件

### Header (Navbar)

| 區域 | 內容                                                         |
| ---- | ------------------------------------------------------------ |
| 左側 | 品牌名稱 **Builder 101**                                     |
| 右側 | 首頁 · 探索項目 · 如何使用 · 關於我們 · 聯絡我們 · 登入/註冊 |

### Footer

| 區塊     | 內容                                   |
| -------- | -------------------------------------- |
| 快速連結 | 探索項目、如何使用、關於我們、聯絡我們 |
| 社群連結 | Facebook、Instagram、Line              |
| 版權宣告 | © 2024 Builder 101                     |
| 法律連結 | 隱私權政策、服務條款 → `terms.html`    |
| 回到頂部 | 固定右下角按鈕（滾動 >300px 顯示）     |

### Toast 訊息

| 觸發情境     | 訊息內容                           | 類型    |
| ------------ | ---------------------------------- | ------- |
| 加入收藏     | 「已加入收藏」                     | success |
| 取消收藏     | 「已取消收藏」                     | info    |
| 聯絡表單送出 | 「訊息已送出！我們會盡快回覆您」   | success |
| 登入成功     | 「登入成功！歡迎回來」             | success |
| 註冊成功     | 「註冊成功！歡迎加入 Builder 101」 | success |
